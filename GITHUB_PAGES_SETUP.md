## 📝 GitHub Pages 활성화 단계

### 1단계: 저장소 설정 페이지로 이동

아래 링크를 클릭하여 저장소 설정으로 이동하세요:

🔗 **https://github.com/xorn1120-taegulee/public/settings**

---

### 2단계: Pages 메뉴 선택

왼쪽 사이드바에서 **"Pages"** 메뉴를 클릭합니다.

---

### 3단계: 소스 설정

**Build and deployment** 섹션에서:

1. **Source**: `Deploy from a branch` 선택
2. **Branch**: 
   - Branch: `main` 선택
   - Folder: `/ (root)` 선택
3. **Save** 버튼 클릭

---

### 4단계: 배포 대기

- GitHub가 자동으로 웹사이트를 배포합니다
- 약 1-2분 정도 소요됩니다
- 페이지를 새로고침하면 배포 URL이 표시됩니다

---

## 🔍 배포 상태 확인

배포 진행 상황을 확인하려면:
